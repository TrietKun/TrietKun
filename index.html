Smart Farm IoT

Real-time IoT Monitoring Platform for Smart Agriculture

⚡ DevStation Internship 📅 Jun – Sep 2025 👥 Team Project

OVERVIEW

A comprehensive IoT monitoring application for smart agriculture. Built during my internship at DevStation, this system enables real-time telemetry control of agricultural sensors through a scalable Farm → Zone → Device hierarchy. The ESP32 firmware communicates with ThingsBoard via MQTT for RPC commands and telemetry data streaming.

KEY FEATURES

  • Real-time sensor telemetry dashboard with live data visualization
  • Scalable Farm → Zone → Device management hierarchy
  • ESP32 firmware with MQTT protocol for bidirectional communication
  • ThingsBoard RPC commands for remote device control
  • Alert system for abnormal sensor readings
  • Historical data charts and analytics
  • Multi-farm support with role-based access

TECH STACK
